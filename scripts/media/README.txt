HOW TO USE THE AUTO REEL CREATOR
=================================

1. DROP YOUR MEDIA INTO THE RIGHT FOLDERS:

   media/before/    → Before photos/videos (dirty car, interior mess, etc.)
   media/after/     → After photos/videos  (clean car, spotless interior, etc.)
   media/action/    → Process shots (detailing in progress, spraying, wiping, etc.)

   FILES ARE MATCHED BY SORT ORDER:
   before_01.jpg pairs with after_01.jpg (first before = first after)

2. OR DUMP EVERYTHING IN THIS FOLDER (media/) WITH THESE NAME PREFIXES:
   before_01.jpg, before_02.jpg ...
   after_01.jpg,  after_02.jpg  ...
   action_01.jpg, action_02.jpg ...

3. ADD BACKGROUND MUSIC (optional):
   Drop a .mp3 or .wav file into: scripts/music/
   (Royalty-free music: Pixabay, Free Music Archive, YouTube Audio Library)

4. RUN THE SCRIPT:
   cd Mobile-Detailing-Business
   python scripts/create_reel.py

5. FIND YOUR REEL:
   scripts/output/reel_YYYYMMDD_HHMMSS.mp4

SUPPORTED FORMATS: .jpg .jpeg .png .mp4 .mov

TIPS FOR VIRAL REELS:
- Shoot before shots FIRST (show the mess clearly)
- After shots: bright lighting, multiple angles
- Action shots: close-ups of spraying/wiping look great
- 3-5 before/after pairs = ideal reel length (~35 seconds)
- Upload to Instagram as a Reel, add trending audio in the app
